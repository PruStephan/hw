\begin{para}{Функция расстояния}

$dist = \alpha dist_{categorical} + (1 - \alpha)dist_{numeric}$

$dist_{categorical}$ - количество свойств, которые не совпадают

$dist_{numeric} = euclidean | manhattan$


\end{para}
